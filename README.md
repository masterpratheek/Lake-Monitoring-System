# Lake Monitoring System

A Python-based environmental monitoring system that stores and displays lake water quality information and provides an AI-powered chatbot for educational assistance.

## Features

- View lake information
- Water quality monitoring records
- pH tracking
- Temperature tracking
- AI chatbot assistance
- CSV-based data storage

## Problem Statement

Monitoring lake ecosystems is essential for maintaining environmental sustainability. Manual tracking of water quality parameters is often inefficient and difficult to access.

## Solution

This project provides a simple lake monitoring platform that stores water quality data and allows users to explore lake conditions through an interactive interface.

## Technologies Used

- Python
- CSV file Data Storage

## Sample Parameters

- pH Level
- Water Temperature
- Timeline Records
